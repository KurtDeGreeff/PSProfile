# PSProfile - ChangeLog

* Initial release to the PowerShell Gallery
* Included functions:
  * `Add-PSProfilePlugin`
  * `Add-PSProfileScriptPath`
  * `Add-PSProfileSymbolicLink`
  * `Add-PSProfileVariable`
  * `Get-MyCreds`
  * `Get-PSProfileArguments`
  * `Get-PSProfileImportedCommands`
  * `Get-PSProfileLog`
  * `Import-PSProfile`
  * `Import-PSProfileConfiguration`
  * `Save-PSProfile`
  * `Set-MyCreds`
  * `Set-PSProfileRefreshFrequency`
  * `Set-PSProfileSetting`
  * `Update-PSProfileConfig`
  * `Write-PSProfileLog`
* Included aliases:
  * `Creds` >> `Get-MyCreds`
  * `Load-PSProfile` >> `Import-PSProfile`
  * `Refresh-PSProfile` >> `Update-PSProfileConfig`
