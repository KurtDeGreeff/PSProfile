# Import-PSProfile

## SYNOPSIS
Reloads your PSProfile by running $PSProfile.Load()

## SYNTAX

```
Import-PSProfile [<CommonParameters>]
```

## DESCRIPTION
Reloads your PSProfile by running $PSProfile.Load()

## EXAMPLES

### EXAMPLE 1
```
Import-PSProfile
```

### EXAMPLE 2
```
Load-PSProfile
```
