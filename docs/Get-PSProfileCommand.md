# Get-PSProfileCommand

## SYNOPSIS
Gets the list of commands provided by PSProfile directly.

## SYNTAX

```
Get-PSProfileCommand [[-Command] <String[]>] [<CommonParameters>]
```

## DESCRIPTION
Gets the list of commands provided by PSProfile directly.

## EXAMPLES

### EXAMPLE 1
```
Get-PSProfileCommand
```

Gets the full list of commands provided by PSProfile directly.
