# Test-IfGit

## SYNOPSIS
Tests if the current path is in a Git repo folder and returns the basic details as an object if so.
Useful in prompts when determining current folder's Git status

## SYNTAX

```
Test-IfGit [<CommonParameters>]
```

## DESCRIPTION
Tests if the current path is in a Git repo folder and returns the basic details as an object if so.
Useful in prompts when determining current folder's Git status

## EXAMPLES

### EXAMPLE 1
